    Note: `torch.set_num_interop_threads(1)` limits inter-op parallelism. Combined with `torch.set_num_threads(2)`, training uses at most 2-3 CPU threads for PyTorch ops, leaving the rest for the beamforming pipeline.

    **2. Modify `src/acoustic/training/trainer.py` -- add TorchScript export and confusion matrix:**

    **Confusion matrix**: In the validation epoch section (inside the `with torch.no_grad():` block), after computing `preds = (output >= 0.5).float()`, accumulate confusion matrix counts:

    ```python
    # Confusion matrix accumulators (initialize before the val loop)
    tp = fp = tn = fn = 0

    # Inside the val batch loop, after preds:
    tp += ((preds == 1) & (batch_y == 1)).sum().item()
    fp += ((preds == 1) & (batch_y == 0)).sum().item()
    tn += ((preds == 0) & (batch_y == 0)).sum().item()
    fn += ((preds == 0) & (batch_y == 1)).sum().item()
    ```

    Include tp, fp, tn, fn in the progress callback dict on every epoch:
    ```python
    "tp": int(tp), "fp": int(fp), "tn": int(tn), "fn": int(fn),
    ```

    **TorchScript export**: After the training loop ends (after the `for epoch in range(...)` block), add an export step BEFORE the final return:

    ```python
    # Export to TorchScript deployable format (ROADMAP SC3)
    if early_stopping.best_loss is not None and ckpt_path.exists():
        try:
            export_model = ResearchCNN()
            export_model.load_state_dict(torch.load(str(ckpt_path), weights_only=True))
            export_model.eval()
            jit_path = Path(str(ckpt_path) + ".jit")
            scripted = torch.jit.script(export_model)
            scripted.save(str(jit_path))
            logger.info("Exported TorchScript model to %s", jit_path)
        except Exception:
            logger.exception("TorchScript export failed, state_dict checkpoint still available")
    ```

    This creates the TorchScript file as an ADDITIONAL export alongside the state_dict .pt file. The state_dict remains the primary format per D-08 (classifier factory loads .pt). The TorchScript file is the "deployable format" satisfying ROADMAP SC3.
