    **1. Add `mel_spectrogram_from_segment()` to `src/acoustic/classification/preprocessing.py`:**

    Extract a public utility function from ResearchPreprocessor internals. This function takes a pre-selected segment (not the whole audio) and returns the mel-spectrogram tensor. The function MUST produce identical output to ResearchPreprocessor.process() when given the same segment.

    ```python
    def mel_spectrogram_from_segment(
        segment: np.ndarray,
        config: MelConfig | None = None,
    ) -> torch.Tensor:
        """Convert a pre-extracted audio segment to a normalized mel-spectrogram.

        Args:
            segment: 1-D float32 mono audio, expected length config.segment_samples.
                     Will be zero-padded if shorter, trimmed if longer.
            config: MelConfig with preprocessing parameters.

        Returns:
            Tensor of shape (1, 1, max_frames, n_mels) with values in [0, 1].
        """
    ```

    Implementation: reuse the same MelSpectrogram transform creation as ResearchPreprocessor (same parameters: sample_rate, n_fft, hop_length, n_mels, power=2.0, norm="slaney", mel_scale="slaney", center=True, pad_mode="constant"). Apply _power_to_db, normalize with (S_db + db_range) / db_range, clamp [0, 1], transpose to (frames, n_mels), pad/trim to max_frames, unsqueeze to (1, 1, 128, 64).

    To avoid recreating the MelSpectrogram transform on every call, use a module-level cache dict keyed by MelConfig instance (it's frozen/hashable).

    **2. Create `src/acoustic/training/dataset.py`** with:

    `collect_wav_files(data_root: str | Path, label_map: dict[str, int]) -> tuple[list[Path], list[int]]`:
    - Scan each subdirectory in data_root whose name is a key in label_map.
    - Collect all `.wav` files (case-insensitive) from each matching subdirectory (non-recursive within each label dir).
    - Return parallel lists of (file_paths, labels).
    - Raise ValueError if data_root does not exist or no WAV files found.

    `DroneAudioDataset(Dataset)` (per D-01):
    - `__init__(self, file_paths: list[Path], labels: list[int], mel_config: MelConfig, waveform_aug: WaveformAugmentation | None = None, spec_aug: SpecAugment | None = None)`
    - `__len__` returns len(file_paths)
    - `__getitem__` does:
      1. `sf.read(self._paths[idx], dtype="float32")` — lazy I/O
      2. If multi-channel, average to mono
      3. Random 0.5s segment: `start = random.randint(0, max(0, len(audio) - n))`, `segment = audio[start:start+n]`. If audio shorter than n, zero-pad.
      4. If waveform_aug is not None, apply it to segment
      5. Call `mel_spectrogram_from_segment(segment, self._mel_config)` to get (1, 1, 128, 64) tensor
      6. If spec_aug is not None, apply it to the (1, 128, 64) slice (squeeze batch dim first)
      7. Return `(features.squeeze(0), label_tensor)` — shapes (1, 128, 64) and scalar float32

    `build_weighted_sampler(labels: list[int]) -> WeightedRandomSampler` (per D-02):
    - Count occurrences of each label value.
    - Compute per-sample weight as 1.0 / class_count.
    - Return `WeightedRandomSampler(weights, num_samples=len(labels), replacement=True)`.
