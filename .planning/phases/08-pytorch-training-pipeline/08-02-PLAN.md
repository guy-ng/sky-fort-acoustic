    **1. Create `src/acoustic/training/manager.py`** with:

    `TrainingStatus(str, Enum)` (per D-13):
    - Values: `IDLE = "idle"`, `RUNNING = "running"`, `COMPLETED = "completed"`, `CANCELLED = "cancelled"`, `FAILED = "failed"`

    `TrainingProgress` dataclass (per D-13):
    - Fields: `status: TrainingStatus = TrainingStatus.IDLE`, `epoch: int = 0`, `total_epochs: int = 0`, `train_loss: float = 0.0`, `val_loss: float = 0.0`, `val_acc: float = 0.0`, `best_val_loss: float = float("inf")`, `error: str | None = None`

    `TrainingManager` class (per D-12, D-13, D-14, D-15):
    - `__init__(self, config: TrainingConfig | None = None, mel_config: MelConfig | None = None)`
    - `self._lock = threading.Lock()`
    - `self._progress = TrainingProgress()`
    - `self._stop_event = threading.Event()`
    - `self._thread: threading.Thread | None = None`

    - `get_progress(self) -> TrainingProgress`: Acquire lock, return a copy via `dataclasses.replace(self._progress)`.

    - `is_training(self) -> bool`: Return True if thread is alive.

    - `start(self, config: TrainingConfig | None = None) -> None` (per D-15):
      1. If `self._thread is not None and self._thread.is_alive()`, raise `RuntimeError("Training already in progress")`.
      2. Reset: `self._stop_event.clear()`, create fresh `TrainingProgress(status=TrainingStatus.RUNNING, total_epochs=cfg.max_epochs)`.
      3. Create `self._thread = threading.Thread(target=self._run, args=(cfg,), daemon=True)` (per D-12).
      4. `self._thread.start()`.

    - `cancel(self) -> None` (per D-14):
      1. `self._stop_event.set()`
      2. If thread is alive, `self._thread.join(timeout=30)`.

    - `_run(self, config: TrainingConfig) -> None`:
      1. `os.nice(10)` (per D-12 — lower CPU priority).
      2. Try: Create `TrainingRunner(config, self._mel_config)`, call `runner.run(self._stop_event, progress_callback=self._on_progress)`.
      3. On completion: update progress status to COMPLETED (or CANCELLED if stop_event was set).
      4. On exception: update progress status to FAILED, store error message.

    - `_on_progress(self, update: dict) -> None`: Acquire lock, update self._progress fields from dict.
