    2. Create src/acoustic/beamforming/bandpass.py with class BandpassFilter:
       - __init__(self, fs: int, fmin: float = 500.0, fmax: float = 4000.0, order: int = 4):
         - Compute Nyquist: nyq = fs / 2.0
         - Compute normalized cutoffs: low = fmin / nyq, high = fmax / nyq
         - Design filter: self._sos = butter(order, [low, high], btype='band', output='sos')
         - Init state tracking: self._zi: np.ndarray | None = None, self._n_channels: int = 0
       - reset(self, n_channels: int) -> None:
         - Get initial conditions: zi = sosfilt_zi(self._sos) — shape (n_sections, 2)
         - Broadcast to per-channel: self._zi = np.repeat(zi[np.newaxis, :, :], n_channels, axis=0)
         - Store self._n_channels = n_channels
       - apply(self, signals: np.ndarray) -> np.ndarray:
         - signals shape: (n_mics, n_samples)
         - If self._zi is None or signals.shape[0] != self._n_channels: call self.reset(signals.shape[0])
         - Allocate output: filtered = np.empty_like(signals)
         - Loop per channel: filtered[ch], self._zi[ch] = sosfilt(self._sos, signals[ch], zi=self._zi[ch])
         - Return filtered

       Imports: numpy as np, from scipy.signal import butter, sosfilt, sosfilt_zi
