    1. Create src/acoustic/beamforming/mcra.py with class MCRANoiseEstimator:
       - __init__(self, alpha_s=0.8, alpha_d=0.95, delta=5.0, min_window=50):
         - Store params as self._alpha_s, self._alpha_d, self._delta, self._min_window
         - Init state: self._frame_count = 0, self._S = None, self._S_min = None, self._noise = None
       - update(self, srp_map: np.ndarray) -> np.ndarray:
         - Flatten: power = srp_map.ravel()
         - First frame (self._S is None): init S, S_min, noise all = power.copy(), frame_count=1, return noise reshaped
         - Subsequent frames:
           - Step 1 Smooth: self._S = self._alpha_s * self._S + (1 - self._alpha_s) * power
           - Step 2 Track min: self._S_min = np.minimum(self._S_min, self._S); increment frame_count; if frame_count % min_window == 0: self._S_min = self._S.copy()
           - Step 3 Signal presence: ratio = self._S / (self._S_min + 1e-10); signal_present = ratio > self._delta
           - Step 4 Conditional update: alpha = np.where(signal_present, self._alpha_d, 0.5); self._noise = alpha * self._noise + (1 - alpha) * power
         - Return self._noise.reshape(srp_map.shape)
       - reset(self) -> None: Set S, S_min, noise to None, frame_count to 0
