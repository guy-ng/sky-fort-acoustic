Create `src/acoustic/training/parquet_dataset.py` with:

1. **`split_indices(total: int, seed: int = 42, train_frac: float = 0.70, val_frac: float = 0.15) -> tuple[list[int], list[int], list[int]]`**
   - Creates list(range(total)), shuffles with `random.Random(seed)`, splits into train/val/test
   - train = indices[:n_train], val = indices[n_train:n_train+n_val], test = indices[n_train+n_val:]
   - Per D-04 (file-level 70/15/15) and D-05 (fixed seed deterministic)

2. **`decode_wav_bytes(wav_bytes: bytes) -> np.ndarray`** (module-level helper)
   - Skip 44-byte WAV header: `np.frombuffer(wav_bytes[44:], dtype=np.int16).astype(np.float32) / 32768.0`
   - Returns 1-D float32 array with values in [-1.0, 1.0]
   - Per D-01 (stream from Parquet, no disk extraction)

3. **`class ParquetDataset(torch.utils.data.Dataset)`**
   - `__init__(self, data_dir: str | Path, split_indices: list[int], mel_config: MelConfig | None = None, waveform_aug: WaveformAugmentation | None = None, spec_aug: SpecAugment | None = None)`
   - Constructor scans `sorted(Path(data_dir).glob("train-*.parquet"))` for shards
   - Reads only `columns=["label"]` from each shard to build `_all_labels` list and `_shard_offsets` cumulative offset list
   - Stores `_indices = split_indices` and `_labels = [_all_labels[i] for i in split_indices]`
   - Per D-02 (separate class from DroneAudioDataset), D-03 (no caching), D-09 (configurable data_dir)
   
   - `_locate(self, global_idx: int) -> tuple[int, int]`: Uses `bisect.bisect_right(self._shard_offsets, global_idx) - 1` to find shard, `local_idx = global_idx - self._shard_offsets[shard_idx]`
   
   - `__len__(self) -> int`: Returns `len(self._indices)`
   
   - `__getitem__(self, idx: int) -> tuple[torch.Tensor, torch.Tensor]`:
     - Maps idx to global_idx via `self._indices[idx]`
     - Calls `_locate(global_idx)` to get shard_idx, local_idx
     - Reads single row: `pq.read_table(self._shards[shard_idx], columns=["audio"])`, then `table.column("audio")[local_idx].as_py()["bytes"]`
     - Decodes with `decode_wav_bytes(wav_bytes)`
     - Random segment extraction: if `len(audio) >= n` (n=mel_config.segment_samples=8000), pick random start; else zero-pad (same logic as DroneAudioDataset)
     - Optional `self._waveform_aug(segment)` if provided
     - `mel_spectrogram_from_segment(segment, self._mel_config).squeeze(0)` -> (1, 128, 64)
     - Optional `self._spec_aug(features)` if provided
     - Returns (features, torch.tensor(self._labels[idx], dtype=torch.float32))
     - Per D-06 (binary labels: 1->1.0, 0->0.0)
   
   - `@property labels(self) -> list[int]`: Returns `self._labels` (needed by `build_weighted_sampler`)
   
   - `@property total_rows(self) -> int`: Returns total rows across all shards (before splitting)

4. **`class ParquetDatasetBuilder`** (lightweight factory):
   - `__init__(self, data_dir: str | Path)`: Scans shards, reads label column, stores total_rows and all_labels
   - `build(self, split_indices: list[int], mel_config=None, waveform_aug=None, spec_aug=None) -> ParquetDataset`: Creates ParquetDataset without re-scanning shards (passes pre-computed shard info)
   - This avoids scanning shards 3 times (once per train/val/test split)

Then create `tests/unit/test_parquet_dataset.py` with test fixtures using small synthetic Parquet files (2-3 shards, 10-20 rows each) created via pyarrow in a pytest fixture. Tests cover all behaviors listed above. Use `tmp_path` fixture for synthetic data directory.
