    **2. Create `src/acoustic/training/efficientat_trainer.py`:**

    Class `EfficientATTrainingRunner` with `run(stop_event, progress_callback) -> Path | None`:

    - **Model setup**: Call `get_model(num_classes=527, pretrained_name=None, width_mult=1.0, head_type="mlp", input_dim_f=128, input_dim_t=config.input_dim_t_or_default_100)`. Load pretrained weights from `config.pretrained_weights` using `torch.load(path, map_location="cpu", weights_only=True)` then `model.load_state_dict(state_dict)`. Call `model.float()` to ensure float32. Replace head: `model.classifier[-1] = nn.Linear(model.classifier[-1].in_features, 1)`.

    - **Data loading**: Reuse the Parquet/WAV data source selection logic from `TrainingRunner.run()`. Audio must be resampled from 16kHz to 32kHz using `torchaudio.functional.resample(waveform, 16000, 32000)`. Create a custom dataset wrapper or modify the preprocessing to handle 32kHz. Use `AugmentMelSTFT` from vendored preprocess.py for mel computation during training (set `freqm=48, timem=192` for SpecAugment during training, `freqm=0, timem=0` for validation).

    - **Loss**: `nn.BCEWithLogitsLoss()` (model outputs raw logits, no sigmoid in forward pass).

    - **Stage 1 (head only)**:
      ```python
      for p in model.parameters(): p.requires_grad = False
      for p in model.classifier.parameters(): p.requires_grad = True
      optimizer = Adam(model.classifier.parameters(), lr=config.stage1_lr)  # 1e-3
      ```
      Train for `config.stage1_epochs` (default 10). Use `EarlyStopping(patience=config.patience)`.

    - **Stage 2 (head + last 3 blocks)**:
      ```python
      for block in model.features[-3:]:
          for p in block.parameters(): p.requires_grad = True
      trainable = [p for p in model.parameters() if p.requires_grad]
      optimizer = Adam(trainable, lr=config.stage2_lr)  # 1e-4
      scheduler = CosineAnnealingLR(optimizer, T_max=config.stage2_epochs)
      ```
      Train for `config.stage2_epochs` (default 15). Call `scheduler.step()` after each epoch.

    - **Stage 3 (all layers)**:
      ```python
      for p in model.parameters(): p.requires_grad = True
      optimizer = Adam(model.parameters(), lr=config.stage3_lr)  # 1e-5
      scheduler = CosineAnnealingLR(optimizer, T_max=config.stage3_epochs)
      ```
      Train for `config.stage3_epochs` (default 20). Call `scheduler.step()` after each epoch.

    - **Progress callback**: Call `progress_callback(dict)` each epoch with keys: `epoch`, `total_epochs` (sum of all 3 stage epochs), `train_loss`, `val_loss`, `val_acc`, `best_val_loss`, `stage` (1/2/3), `stage_epoch`.

    - **Checkpoint saving**: Save best model (lowest val_loss across all stages) to `config.checkpoint_path`. Save full model state_dict (not just head). Use `torch.save(model.state_dict(), path)`.

    - **Stop event**: Check `stop_event.is_set()` at each epoch boundary. If set, return best checkpoint path or None.
