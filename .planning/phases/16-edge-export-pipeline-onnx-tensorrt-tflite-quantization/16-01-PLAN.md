    ```python
    class ExportPipeline:
        """Orchestrates model export from PyTorch to ONNX and optional downstream formats."""

        SUPPORTED_MODELS = {"research_cnn", "efficientat"}
        SUPPORTED_FORMATS = {"onnx", "tensorrt", "tflite"}

        def export_to_onnx(
            self,
            checkpoint_path: str,
            output_path: str,
            model_type: str,  # "research_cnn" or "efficientat"
        ) -> dict:
    ```

    Implementation:
    1. Validate `model_type` in SUPPORTED_MODELS, raise ValueError if not.
    2. Validate checkpoint_path exists, raise FileNotFoundError if not.
    3. If model_type == "research_cnn": call `prepare_research_cnn(checkpoint_path)`, set `dummy_input = torch.randn(1, 1, 128, 64)`, input_name="mel_spectrogram", output_name="probability".
    4. If model_type == "efficientat": call `prepare_efficientat(checkpoint_path)`, set `dummy_input = torch.randn(1, 1, 128, 100)` (from EfficientATMelConfig: n_mels=128, input_dim_t=100), input_name="mel_spectrogram", output_name="logits".
    5. Call `torch.onnx.export(model, dummy_input, output_path, opset_version=17, input_names=[input_name], output_names=[output_name], dynamic_axes={input_name: {0: "batch"}, output_name: {0: "batch"}}, do_constant_folding=True)`.
    6. Call `validate_onnx_model(output_path)`.
    7. Call `parity = validate_onnx_parity(model, output_path, dummy_input, input_name=input_name)`.
    8. Return `{"status": "success" if parity["parity_ok"] else "parity_failed", "path": output_path, "max_diff": parity["max_diff"], "opset": 17, "model_type": model_type}`.

    ```python
        ) -> dict:
    ```
