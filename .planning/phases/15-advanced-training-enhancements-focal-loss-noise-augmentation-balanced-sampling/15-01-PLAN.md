    1. Add `BackgroundNoiseMixer` class to `src/acoustic/training/augmentation.py`:
       ```python
       class BackgroundNoiseMixer:
           """Mix background noise from ESC-50/UrbanSound8K at random SNR (TRN-11)."""

           def __init__(
               self,
               noise_dirs: list[Path],
               snr_range: tuple[float, float] = (-10.0, 20.0),
               sample_rate: int = 16000,
               p: float = 0.5,
           ) -> None:
       ```
       - Constructor scans each directory in `noise_dirs` for `*.wav` files recursively using `rglob("*.wav")`
       - Stores file list but does NOT load audio yet (lazy via `warm_cache()`)
       - `warm_cache()` loads all noise WAV files into memory as float32 numpy arrays, resampling to `sample_rate` using `torchaudio.functional.resample` if source SR differs. Converts stereo to mono via `.mean(axis=1)`.
       - `__call__(audio: np.ndarray) -> np.ndarray`:
         - With probability `1 - p`, return audio unchanged
         - If `_noise_cache` is empty, return audio unchanged
         - Pick random noise clip from cache
         - Extract segment matching `len(audio)`: if noise longer, pick random start; if shorter, zero-pad
         - Compute scale factor: `scale = sqrt(P_signal / (P_noise * 10^(SNR_dB / 10)))`
         - Mix: `audio + scale * noise_segment`
         - Clip output to `[-1.0, 1.0]` to prevent overflow
         - Return as float32
       - Uses `np.random.default_rng()` stored as `self._rng`
