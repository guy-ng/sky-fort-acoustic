    **src/acoustic/pipeline.py** — Background beamforming processor:
    - `class BeamformingPipeline`:
      - `__init__(self, settings: AcousticSettings)`:
        - Store settings
        - Build mic_positions via build_mic_positions()
        - Build az_grid_deg = np.arange(-settings.az_range, settings.az_range + settings.az_resolution, settings.az_resolution)
        - Build el_grid_deg = np.arange(-settings.el_range, settings.el_range + settings.el_resolution, settings.el_resolution)
        - Init latest_map: np.ndarray | None = None
        - Init latest_peak: PeakDetection | None = None
        - Init _running: bool = False
        - Init _thread: threading.Thread | None = None
        - Init _last_process_time: float | None = None (monotonic timestamp of last successful process_chunk)
      - `process_chunk(self, chunk: np.ndarray) -> PeakDetection | None`:
        - chunk is (chunk_samples, num_channels) from ring buffer
        - Transpose to (num_channels, chunk_samples) for SRP-PHAT
        - Call srp_phat_2d(signals, self.mic_positions, settings.sample_rate, settings.speed_of_sound, self.az_grid_deg, self.el_grid_deg, settings.freq_min, settings.freq_max)
        - Store result in self.latest_map
        - Call detect_peak_with_threshold(map, az_grid, el_grid, settings.noise_percentile, settings.noise_margin)
        - Store result in self.latest_peak
        - Set self._last_process_time = time.monotonic()
        - Return the peak (or None)
      - `start(self, ring_buffer: AudioRingBuffer)`: Start a daemon thread that loops: read chunk from ring, if chunk is not None call process_chunk, else sleep 0.01s. Set _running = True.
      - `stop(self)`: Set _running = False, join thread.
      - `@property running -> bool`: return _running
      - `@property last_process_time -> float | None`: return _last_process_time

      7. pipeline.start(capture.ring)
      8. Store on app.state: settings, device_info, capture, pipeline
      9. yield
      10. pipeline.stop()

    ```python
    def test_pipeline_liveness(settings):
        """Verify pipeline.latest_map updates over time (not stalled)."""
        pipeline = BeamformingPipeline(settings)
        ring = AudioRingBuffer(settings.ring_chunks, settings.chunk_samples, settings.num_channels)
        sim = SimulatedAudioSource(settings)
        # Start a producer feeding chunks
        producer = SimulatedProducer(sim, ring, settings.chunk_seconds)
        producer.start()
        pipeline.start(ring)
        try:
            time.sleep(0.5)  # Let pipeline process a few chunks
            map_1 = pipeline.last_process_time
            assert map_1 is not None, "Pipeline never processed a chunk"
            time.sleep(0.3)
            map_2 = pipeline.last_process_time
            assert map_2 > map_1, "Pipeline stalled — last_process_time not advancing"
        finally:
            pipeline.stop()
            producer.stop()
    ```
