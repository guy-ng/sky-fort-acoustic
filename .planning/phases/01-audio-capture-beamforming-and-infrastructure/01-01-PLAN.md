    **src/acoustic/audio/capture.py** — Implement per D-01, D-02, D-03 and RESEARCH Pattern 1:
    - `class AudioRingBuffer`:
      - `__init__(self, num_chunks: int, chunk_samples: int, num_channels: int)`: Pre-allocate `np.zeros((num_chunks, chunk_samples, num_channels), dtype=np.float32)`. Initialize write_idx=0, read_idx=0, _overflow_count=0.
      - `write(self, data: np.ndarray) -> bool`: Copy data into buffer[write_idx]. Advance write_idx. Return False if full (next_idx == read_idx), incrementing _overflow_count.
      - `read(self) -> np.ndarray | None`: Return copy of buffer[read_idx] or None if empty. Advance read_idx.
      - `available` property: return (write_idx - read_idx) % num_chunks
      - `overflow_count` property: return _overflow_count

    - `class AudioCapture`:
      - `__init__(self, device: str | int | None, fs: int, channels: int, chunk_samples: int, ring_chunks: int = 14)`: Create AudioRingBuffer. Create sd.InputStream with device=device, samplerate=fs, channels=channels, dtype='float32', blocksize=chunk_samples, callback=self._callback. Store `_last_frame_time = None`.
      - `_callback(self, indata, frames, time_info, status)`: Only np.copyto or array assignment — NO logging, NO GIL-heavy operations (per Pitfall 1). Set self._last_frame_time = time.monotonic() before write. If status: set self._xrun_flag = True.
      - `start(self)` / `stop(self)`: Start/stop and close the stream.
      - `@property ring`: expose the ring buffer.
      - `@property last_frame_time`: return _last_frame_time.
