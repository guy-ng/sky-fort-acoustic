    3. **Model Registry** (module-level):
       - `ModelLoader = Callable[[str], Classifier]` type alias
       - `_REGISTRY: dict[str, ModelLoader] = {}` module-level dict
       - `register_model(type_name: str, loader: ModelLoader) -> None`: adds to `_REGISTRY`
       - `load_model(type_name: str, path: str) -> Classifier`: looks up `_REGISTRY[type_name]`, raises ValueError if not found with message f"Unknown model type: {type_name}. Registered: {list(_REGISTRY)}"

       - `__init__(self, classifiers: list[Classifier], weights: list[float], *, max_live_models: int = 3, live_mode: bool = True) -> None`
       - Validates: len(classifiers) == len(weights), raises ValueError "classifiers and weights must have same length" if not
       - Validates: if live_mode and len(classifiers) > max_live_models, raises ValueError f"Live mode allows max {max_live_models} models, got {len(classifiers)}"
       - Normalizes weights: `total = sum(weights); self._weights = [w / total for w in weights]`
       - `predict(self, features: torch.Tensor) -> float`: iterates classifiers/weights, returns `sum(w * clf.predict(features) for clf, w in zip(...))`
       - `@property model_count(self) -> int`: returns len(self._classifiers)
       - `@property weights(self) -> list[float]`: returns copy of normalized weights
