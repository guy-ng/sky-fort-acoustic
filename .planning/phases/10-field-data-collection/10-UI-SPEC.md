### Layout Integration

Recording UI occupies the **sidebar column** (right side, `gridArea: 'sidebar'`), replacing or stacking below the existing Sidebar. The sidebar column is 300px wide ‚Äî sufficient for recording controls and a compact recordings list.

Alternative: Add a **fourth grid row** below the current layout for a full-width recording strip. This is less disruptive to the existing layout.

**Recommended approach:** Add recording controls as a collapsible section within the sidebar panel, below the existing SYSTEM stats. When recording is active, the recording section expands to show timer + level meter. When idle, it shows a compact "Start Recording" button and unlabeled count badge. Recordings list is a separate panel that replaces the sidebar content when toggled via a tab/section switcher in the sidebar header.

No router required ‚Äî sidebar switches between "SYSTEM" and "RECORDINGS" views via local state.

## Interaction States

### Recording Button

| State | Visual | Label |
|-------|--------|-------|
| Idle (device connected) | Solid `hud-accent` background, white text | "Start Recording" |
| Idle (device disconnected) | Solid `hud-border` background, `hud-text-dim` text, `cursor-not-allowed` | "No Device" |
| Recording | Pulsing `hud-danger` background (CSS `animate-pulse`), white text | "Stop Recording" |
| Saving (after stop, before label) | Solid `hud-warning` background, dark text | "Saving..." (disabled) |

### Inline Label Form (appears after recording stops, per D-04)

| Element | Type | Details |
|---------|------|---------|
| Top-level label | 3-button radio group | "drone" / "background" / "other" ‚Äî required, `hud-accent` border on selected |
| Sub-label | Text input with datalist suggestions | Placeholder: "e.g., Mavic 3, traffic, wind" ‚Äî optional |
| Distance | Number input | Placeholder: "meters" ‚Äî optional, `type="number"`, `min=0` |
| Altitude | Number input | Placeholder: "meters" ‚Äî optional, `type="number"`, `min=0` |
| Conditions | Text input | Placeholder: "e.g., light wind, rain" ‚Äî optional |
| Notes | Textarea (2 rows) | Placeholder: "Free-text notes" ‚Äî optional |
| Save button | Button | "Save Recording" ‚Äî `hud-accent` background, disabled until top-level label selected |
| Discard button | Text button | "Discard" ‚Äî `hud-text-dim`, no background |

Form inputs: `bg-hud-bg`, `border border-hud-border`, `rounded`, `px-2 py-1.5`, `text-sm text-hud-text`, `focus:border-hud-accent focus:outline-none`. Consistent with HUD dark theme.

### Audio Level Meter

| Property | Value |
|----------|-------|
| Shape | Horizontal bar, full width of recording section, height 6px (`h-1.5`) |
| Background | `hud-border` (`#1f2937`) |
| Fill | Gradient: `hud-success` (low) to `hud-warning` (mid) to `hud-danger` (high/clipping) |
| Update rate | 10 Hz (matched to `/ws/recording` poll rate from RESEARCH.md) |
| Range | -60 dB to 0 dB, linear-mapped to bar width |

### Recordings List

| Element | Visual |
|---------|--------|
| Row (normal) | `bg-hud-panel`, `border-b border-hud-border`, `py-2 px-3` |
| Row (unlabeled) | Same + left border `border-l-2 border-hud-warning` |
| Row (selected/editing) | Same + left border `border-l-2 border-hud-accent` |
| Label badge | Inline pill: `text-xs`, `px-1.5 py-0.5`, `rounded`, bg varies by label |
| Label badge: drone | `bg-hud-danger/20 text-hud-danger` |
| Label badge: background | `bg-hud-success/20 text-hud-success` |
| Label badge: other | `bg-hud-warning/20 text-hud-warning` |
| Label badge: unlabeled | `bg-hud-warning/20 text-hud-warning`, italic text |
| Duration | `font-mono text-xs text-hud-text-dim` |
| Timestamp | `font-mono text-xs text-hud-text-dim` |
| Delete icon | Material Symbol `delete`, `text-hud-text-dim hover:text-hud-danger`, 20px |
| Edit icon | Material Symbol `edit`, `text-hud-text-dim hover:text-hud-accent`, 20px |

### Timer Display (during recording)

| Property | Value |
|----------|-------|
| Format | `MM:SS` elapsed / `MM:SS` remaining (or just elapsed if no max) |
| Font | JetBrains Mono, `text-2xl`, `font-semibold` |
| Color | `hud-text` normally, `hud-warning` when <30s remaining, `hud-danger` when <10s remaining |
