    **REST endpoints** (`src/acoustic/api/recording_routes.py`):
    Create `router = APIRouter(prefix="/api/recordings", tags=["recordings"])` following training_routes.py pattern.

    Endpoints:
    - `POST /api/recordings/start` -> `RecordingStartResponse`
      Calls `manager.start_recording()`. Returns 200 with id and status="recording". Returns 409 if already recording. Returns 503 if no device.
    - `POST /api/recordings/stop` -> `RecordingStopResponse`
      Calls `manager.stop_recording()`. Returns 200 with id and duration. Returns 409 if not recording.
    - `GET /api/recordings` -> `list[dict]`
      Calls `manager.list_recordings()`. Returns 200 with array of recording objects.
    - `GET /api/recordings/{rec_id}` -> `dict`
      Calls `manager.get_recording(rec_id)`. Returns 200 or 404.
    - `POST /api/recordings/{rec_id}/label` -> `dict` (accepts `LabelRequest` body)
      Calls `manager.label_recording(rec_id, body.label, extra=body.model_dump(exclude={"label"}, exclude_none=True))`. Returns 200 or 404/400.
    - `PATCH /api/recordings/{rec_id}` -> `dict` (accepts `MetadataUpdateRequest` body)
      Calls `manager.update_recording(rec_id, body.model_dump(exclude_none=True))`. Returns 200 or 404.
    - `DELETE /api/recordings/{rec_id}` -> `RecordingDeleteResponse`
      Calls `manager.delete_recording(rec_id)`. Returns 200 or 404.

    Access manager via `request.app.state.recording_manager`.

    **WebSocket** (add to existing `src/acoustic/api/websocket.py`):
    Add `/ws/recording` endpoint following `/ws/training` pattern but at 10Hz (100ms sleep) for responsive level meter per D-03 and RESEARCH.md:
    ```python
    @router.websocket("/ws/recording")
    async def ws_recording(websocket: WebSocket) -> None:
        await websocket.accept()
        manager = websocket.app.state.recording_manager
        last_state: dict | None = None
        try:
            while True:
                state = manager.get_state()
                if state != last_state:
                    await websocket.send_json(state)
                    last_state = state
                await asyncio.sleep(0.1)  # 10Hz for level meter
        except (WebSocketDisconnect, RuntimeError):
            pass
    ```
