    ```python
    @dataclass
    class ClassificationResult:
        drone_probability: float
        timestamp: float
        az_deg: float
        el_deg: float
    ```

    `class CNNWorker`:
    - `__init__(self, classifier: OnnxDroneClassifier, fs_in: int = 48000)` — stores classifier, creates `queue.Queue(maxsize=1)`, threading.Lock for latest result, threading.Event for stop signal, daemon thread.
    - `start(self) -> None` — starts the background thread.
    - `stop(self) -> None` — sets stop event, joins thread with 5s timeout.
    - `push(self, mono_audio: np.ndarray, az_deg: float, el_deg: float) -> None` — non-blocking put. Uses `put_nowait()`, catches `queue.Full` (drop semantics — latest-only). The mono_audio is raw mono at `fs_in` sample rate.
    - `get_latest(self) -> ClassificationResult | None` — returns latest result under lock.
    - `_loop(self) -> None` — while not stopped: get from queue (timeout=0.1), call `preprocess_for_cnn(mono_audio, self._fs_in)`, call `self._classifier.predict(preprocessed)`, store as `ClassificationResult` under lock. Log inference time at DEBUG level.

    **Add `/ws/events` endpoint** (per D-08, D-10):
    ```python
    @router.websocket("/ws/events")
    async def ws_events(websocket: WebSocket) -> None:
        """Stream detection events (new, update, lost) to external consumers.

        Separate from /ws/targets which streams current target state for the UI.
        This endpoint broadcasts detection lifecycle events as they occur.
        """
        await websocket.accept()
        broadcaster: EventBroadcaster | None = getattr(websocket.app.state, "event_broadcaster", None)
        if broadcaster is None:
            # CNN not initialized — close with reason
            await websocket.close(code=1011, reason="Event broadcasting not available")
            return

        queue = broadcaster.subscribe()
        try:
            while True:
                event_data = await queue.get()
                await websocket.send_json(event_data)
        except (WebSocketDisconnect, RuntimeError):
            logger.debug("Events WebSocket client disconnected")
        finally:
            broadcaster.unsubscribe(queue)
    ```
    - Import `EventBroadcaster` from `acoustic.tracking.events` at top of file.
