    **5. Create `src/acoustic/classification/inference.py`:**
    Class `OnnxDroneClassifier`:
    - `__init__(self, model_path: str)` -- loads ONNX model via `ort.InferenceSession(model_path, providers=["CPUExecutionProvider"])`. Stores input name from `session.get_inputs()[0].name`. Raises `FileNotFoundError` if path doesn't exist (check before loading).
    - `predict(self, preprocessed: np.ndarray) -> float` -- runs `session.run(None, {input_name: preprocessed})`, returns `float(outputs[0][0, 0])`. Input must be (1, 128, 64, 1) float32.
    - `predict_from_audio(self, mono_audio: np.ndarray, fs_in: int) -> float` -- calls `preprocess_for_cnn(mono_audio, fs_in)` then `self.predict()`. Convenience method for pipeline integration.

    **6. Create `src/acoustic/classification/state_machine.py`:**
    Enum `DetectionState(str, Enum)` with values: `NO_DRONE = "NO_DRONE"`, `CANDIDATE = "DRONE_CANDIDATE"`, `CONFIRMED = "DRONE_CONFIRMED"`.
    Class `DetectionStateMachine`:
    - `__init__(self, enter_threshold: float = 0.80, exit_threshold: float = 0.40, confirm_hits: int = 2)`
    - `self.state = DetectionState.NO_DRONE`, `self._hit_count = 0`
    - `update(self, drone_probability: float) -> DetectionState` implementing:
      - NO_DRONE + prob >= enter -> CANDIDATE, hit_count=1
      - CANDIDATE + prob >= enter -> hit_count+=1, if hit_count >= confirm_hits -> CONFIRMED
      - CANDIDATE + prob <= exit -> NO_DRONE, hit_count=0
      - CANDIDATE + exit < prob < enter -> stay CANDIDATE (no change)
      - CONFIRMED + prob <= exit -> CANDIDATE, hit_count=0
      - CONFIRMED + prob > exit -> stay CONFIRMED
    - `reset(self) -> None` -- resets to NO_DRONE, hit_count=0
