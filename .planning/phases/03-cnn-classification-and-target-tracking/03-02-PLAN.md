    ```python
    from typing import Literal

    class TargetEvent(BaseModel):
        event: Literal["new", "update", "lost"]
        target_id: str      # UUID string
        class_label: str    # "drone" or "background"
        confidence: float   # 0.0-1.0

    ```

    **3. Create `src/acoustic/tracking/tracker.py`:**
    Dataclass `TrackedTarget`:
    ```python
    @dataclass
    class TrackedTarget:
        target_id: str          # UUID4 string, assigned on creation
        class_label: str        # "drone" or "background"
        confidence: float       # latest drone probability
        az_deg: float           # latest azimuth
        el_deg: float           # latest elevation
        speed_mps: float | None # always None in this phase (D-07)
        first_seen: float       # time.monotonic()
        last_seen: float        # time.monotonic()
    ```

    Class `TargetTracker`:
    - `__init__(self, lost_timeout: float = 5.0)` -- `self._targets: dict[str, TrackedTarget] = {}`, `self._lost_timeout = lost_timeout`
    - `update_target(self, class_label: str, confidence: float, az_deg: float, el_deg: float) -> tuple[str, TrackedTarget]`:
      - If no active targets: create new TrackedTarget with `uuid.uuid4()` as target_id, return `("new", target)`
      - If active target exists: update its confidence, az_deg, el_deg, last_seen. Return `("update", target)`
      - For Phase 3, single-target tracking only (per RESEARCH open question #2). If a target exists, update it rather than creating a new one. Multi-target deferred.
    - `expire_lost_targets(self) -> list[TrackedTarget]`:
      - Check all targets: if `time.monotonic() - target.last_seen > self._lost_timeout`, remove from active and return as lost list
      - Per D-06: 5-second TTL after last detection signal
    - `get_active_targets(self) -> list[TrackedTarget]`:
      - Return list of all active (non-expired) targets
    - `get_target_states(self) -> list[dict]`:
      - Return list of dicts matching TargetState API model fields: `{"id": target_id, "class_label": ..., "speed_mps": None, "az_deg": ..., "el_deg": ..., "confidence": ...}`
    - `clear(self) -> None` -- remove all targets

    NOTE: TRK-02 (Doppler speed estimation) deferred to ms-2 per D-07, D-12. speed_mps always None.

    **4. Create `src/acoustic/tracking/publisher.py`:**
    Class `ZmqPublisher`:
    - `__init__(self, endpoint: str = "tcp://*:5556", sndhwm: int = 100)`:
      - Create `zmq.Context.instance()`, create `zmq.PUB` socket
      - Set `zmq.SNDHWM` to sndhwm (per Research Pitfall #4 -- prevent slow subscriber blocking)
      - Bind to endpoint
      - Set `self._topic = b"acoustic/targets"` (per D-08)
    - `publish(self, event: TargetEvent) -> None`:
      - Serialize event to JSON: `event.model_dump_json().encode("utf-8")`
      - Send multipart: `[self._topic, payload]` with `zmq.NOBLOCK` flag
      - Catch `zmq.Again` silently (no subscribers connected = OK)
    - `close(self) -> None`:
      - Close socket (do NOT terminate context -- it's shared)
    - Per D-10: PUB/SUB pattern, publisher binds, subscribers connect.
