# Phase 20.1 Plan 01: Acquire Noise Corpora — Acquisition Script Summary

**One-liner:** Operator-runnable Python script that downloads ESC-50 + UrbanSound8K + an FSD50K 6-class subset via soundata 1.0 with disk guard, idempotency marker, path-traversal hardening, and 11 passing unit tests.

**Task 1 (autonomous): COMPLETE** — script + dep + tests committed in `4d137bc`.

## What Was Built

### `scripts/acquire_noise_corpora.py`
- **CLI:** `argparse` with subcommands `esc50`, `urbansound8k`, `fsd50k`, `all`; flags `--force`, `--data-dir`.
- **Idempotency:** `.acquired.json` marker (schema_version=1) at `data/noise/<corpus>/`. Re-running is a no-op when `_count_wavs(corpus_dir) == marker.file_count`.
- **Disk guard:** `check_disk_space()` enforces per-corpus thresholds (esc50=2 GB, urbansound8k=14 GB, fsd50k=30 GB) before any download. Raises `RuntimeError` with the actual free GB / required GB.
- **FSD50K class filter (D-09):** Pass 1 — primary-tag rule (`labels.split(",", 1)[0].strip()`); Pass 2 — secondary-label fallback for under-populated classes (<100 files), most likely Mechanical_fan. Logs a WARNING when fallback fires.
- **Path-traversal guard (T-20.1-03):** `_safe_copy()` resolves the destination and asserts `.is_relative_to(NOISE_ROOT.resolve())` before any `shutil.copy`. Refuses `../../etc/passwd` style fnames.
- **Validation:** `soundata.validate()` is called after every `download()`; aborts on any missing file or invalid checksum.
- **Staging cleanup:** FSD50K stages into `data/noise/.fsd50k_stage/` and `shutil.rmtree()`'s it inside a `try/finally` so a mid-download crash never leaves 24 GB orphaned.
- **Lazy soundata import** (deviation, see below): `--help` works on hosts that have not yet `pip install -r requirements.txt`'d; `acquire_*` raise a clean `RuntimeError("soundata is not installed. Run: pip install -r requirements.txt")`.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking issue] soundata not installed in dev venv → made import lazy**
- **Found during:** Task 1 acceptance check (`python scripts/acquire_noise_corpora.py --help`)
- **Issue:** The plan had `import soundata` at module top. The dev venv on this machine does not have `soundata` installed (the operator installs it on the training host as part of Task 2). With a top-level import, even `--help` errored with `ModuleNotFoundError`, which fails acceptance criterion "`python scripts/acquire_noise_corpora.py --help` exits 0".
- **Fix:** Wrapped the import in `try/except ImportError` setting `soundata = None`, then added `_require_soundata()` helper that raises a clear `RuntimeError("soundata is not installed. Run: pip install -r requirements.txt")` at the start of every `acquire_*()` function. Each acquire function captures the local reference (`sd = _require_soundata()`) and calls `sd.initialize(...)` so monkeypatching `ans.soundata` in tests still routes through the same module attribute.
- **Files modified:** `scripts/acquire_noise_corpora.py`
- **Why this is better than the plan as written:** Operators get an actionable error message ("Run: pip install ...") instead of a Python traceback, AND the unit tests can run on any host without the heavy `soundata`+`librosa`+`numba`+`pandas` install chain. The behavioral contract is unchanged: trying to actually acquire data without soundata raises a clean error before any disk or network work.
- **Threat model impact:** None — the lazy import does not weaken T-20.1-01/02 (the validate() call still gates every download path; the dep is still pinned).

**2. [Rule 1 - Bug] Plan-supplied test code used a 3-tuple for `shutil.disk_usage` mock**
- **Found during:** First pytest run (`test_disk_guard_aborts_on_low_space`)
- **Issue:** The plan suggested mocking `shutil.disk_usage` to return a tuple `(total, used, free)`, but `shutil.disk_usage()` returns a `namedtuple` and the script accesses `.free` as an attribute → `AttributeError: 'tuple' object has no attribute 'free'`.
- **Fix:** Both `test_disk_guard_*` tests now build a tiny attribute-bearing object (`type("DiskUsage", (), {})()`) with `.total`, `.used`, `.free` set explicitly. Equivalent semantics to a `namedtuple`, no `collections` import noise.
- **Files modified:** `tests/unit/test_acquire_noise_corpora.py`
- **Commit:** `4d137bc`
